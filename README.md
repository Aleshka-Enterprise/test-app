# test-app
Приложение с тестами.

## Окружение
- python 3.11
- node 16.16.0

# Запуск
## Frontend
```
npm install
npm start
```
## Backend
### Установка
```
pip install -r requirements.txt
```
### Произвести миграции
```
python manage.py migrate
```
### Запуск сервера
```
python manage.py runserver
```
